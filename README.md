# Student Utility Hub

A single frontend web app containing multiple student-focused utilities.

## Setup

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

## Tools

- Resume Builder (live preview, templates, save draft, export PDF)
- PDF Compressor (best-effort client-side re-save)
- PDF Merger
- Notes to PDF
- CGPA Calculator
- Pomodoro Timer
- Password Generator
